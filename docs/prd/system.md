# 시스템 (System)

> 관리자 계정 관리 및 공통 코드 관리 기능을 포함한다.

---

## 관리자 계정

### 로그인
- [ ] 아이디/비밀번호로 로그인할 수 있다
- [ ] 로그인 실패 시 오류 메시지를 표시한다
- [ ] 로그인 성공 시 어드민 메인으로 이동한다
- [ ] 로그아웃할 수 있다

### 관리자 목록 조회
- [ ] 등록된 관리자 목록을 조회할 수 있다
- [ ] 관리자명, 아이디, 역할, 사용여부로 검색할 수 있다

### 관리자 등록
- [ ] 아이디, 비밀번호, 관리자명, 역할(SUPER/ADMIN)을 입력하여 등록할 수 있다
- [ ] 아이디 중복 시 오류 메시지를 표시한다

### 관리자 수정
- [ ] 관리자명, 역할, 사용여부를 수정할 수 있다
- [ ] 비밀번호를 초기화할 수 있다

---

## 공통코드

### 코드 그룹 목록 조회
- [ ] 등록된 코드 그룹 목록을 조회할 수 있다
- [ ] 그룹코드, 그룹명, 사용여부로 검색할 수 있다

### 코드 그룹 등록
- [ ] 그룹코드, 그룹명을 입력하여 등록할 수 있다
- [ ] 그룹코드 중복 시 오류 메시지를 표시한다

### 코드 그룹 수정
- [ ] 그룹명, 사용여부를 수정할 수 있다

### 코드 상세 목록 조회
- [ ] 코드 그룹을 선택하면 하위 코드 목록을 조회할 수 있다

### 코드 상세 등록
- [ ] 코드값, 코드명, 정렬순서를 입력하여 등록할 수 있다
- [ ] 동일 그룹 내 코드값 중복 시 오류 메시지를 표시한다

### 코드 상세 수정
- [ ] 코드명, 정렬순서, 사용여부를 수정할 수 있다
