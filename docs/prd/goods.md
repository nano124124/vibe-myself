# 상품/전시 (Goods & Display)

> 상품 등록/수정/삭제, 카테고리, 브랜드, 옵션, 단품, 재고, 전시 관리 기능을 포함한다.

---

## 카테고리 관리

### 카테고리 목록 조회
- [ ] 카테고리 목록을 트리 구조로 조회할 수 있다
- [ ] 대/중/소 3단계 계층을 지원한다

### 카테고리 등록
- [ ] 카테고리명, 상위 카테고리, 정렬순서를 입력하여 등록할 수 있다

### 카테고리 수정
- [ ] 카테고리명, 정렬순서, 사용여부를 수정할 수 있다

---

## 브랜드 관리

### 브랜드 목록 조회
- [ ] 등록된 브랜드 목록을 조회할 수 있다
- [ ] 브랜드명, 사용여부로 검색할 수 있다

### 브랜드 등록
- [ ] 브랜드명, 브랜드 이미지를 입력하여 등록할 수 있다

### 브랜드 수정
- [ ] 브랜드명, 이미지, 사용여부를 수정할 수 있다

---

## 옵션 관리

> 상품과 독립적으로 존재하며, 카테고리처럼 별도로 등록/관리한다.

### 옵션 그룹 목록 조회
- [ ] 등록된 옵션 그룹 목록을 조회할 수 있다 (예: 색상, 사이즈)
- [ ] 그룹명, 사용여부로 검색할 수 있다

### 옵션 그룹 등록
- [ ] 옵션 그룹명, 정렬순서를 입력하여 등록할 수 있다

### 옵션 그룹 수정
- [ ] 옵션 그룹명, 정렬순서, 사용여부를 수정할 수 있다

### 옵션 항목 목록 조회
- [ ] 옵션 그룹을 선택하면 하위 항목 목록을 조회할 수 있다 (예: 빨강/파랑, S/M/L)

### 옵션 항목 등록
- [ ] 항목명, 정렬순서를 입력하여 등록할 수 있다

### 옵션 항목 수정
- [ ] 항목명, 정렬순서, 사용여부를 수정할 수 있다

---

## 상품 관리

### 상품 목록 조회
- [ ] 등록된 상품 목록을 조회할 수 있다
- [ ] 상품명, 카테고리, 브랜드, 판매상태로 검색할 수 있다

### 상품 등록
- [ ] 상품명, 기본가격, 카테고리, 브랜드를 입력하여 등록할 수 있다
- [ ] 상품 이미지를 최대 5장 업로드할 수 있다
- [ ] 상품 상세 설명을 입력할 수 있다
- [ ] 판매상태(판매중/판매중지)를 설정할 수 있다
- [ ] 판매시작일시, 판매종료일시를 설정할 수 있다
- [ ] 옵션 그룹을 1개 이상 연결할 수 있다
- [ ] 옵션 그룹 연결 시 항목 조합이 단품으로 자동 생성된다

### 상품 수정
- [ ] 상품 기본 정보를 수정할 수 있다
- [ ] 이미지를 추가/삭제/순서 변경할 수 있다
- [ ] 연결된 옵션 그룹을 수정할 수 있다

### 상품 판매 상태 변경
- [ ] 판매상태를 판매중/판매중지로 변경할 수 있다
- [ ] 판매중지는 삭제와 동일한 개념으로 처리한다 (물리적 삭제 없음)

---

## 단품 관리

> 상품에 연결된 옵션 항목 조합이 단품이 된다. (예: 색상=빨강 + 사이즈=S)

### 단품 목록 조회
- [ ] 상품별 단품 목록을 조회할 수 있다
- [ ] 단품별 옵션 조합, 가격, 재고를 확인할 수 있다

### 단품 수정
- [ ] 단품별 추가 가격을 설정할 수 있다
- [ ] 단품별 재고 수량을 수정할 수 있다
- [ ] 단품별 사용여부를 설정할 수 있다

---

## 전시 관리

### 전시 목록 조회
- [ ] 전시 중인 상품 목록을 조회할 수 있다
- [ ] 카테고리, 노출여부로 필터링할 수 있다

### 전시 설정
- [ ] 상품별 노출 여부를 설정할 수 있다
- [ ] 전시 카테고리를 설정할 수 있다
- [ ] 전시 순서를 설정할 수 있다

---

## 쇼핑몰 화면 (고객)

### 상품 목록
- [ ] 카테고리별 상품 목록을 조회할 수 있다
- [ ] 브랜드별 상품 목록을 조회할 수 있다

### 상품 상세
- [ ] 상품 기본 정보 및 이미지를 조회할 수 있다
- [ ] 옵션을 선택할 수 있다
- [ ] 선택한 옵션 조합(단품)의 가격과 재고를 확인할 수 있다
